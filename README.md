Project: Word2Vec Implementation in CWord2vec is a word embedding algorithm in Natural Language Processing that helps to represent the words in a vector space in such a way that words with closer meaning and contextually stands close stay near in the vector space.It implements the word2vec algorithm invented by Google in 2013 from scratch in C code without using any machine learning library.Objectives and FeaturesThe main goal of this project is to represent all the words in the vocabulary in a vector space smoothly to implement the following features:1. Word SimilarityComputes the cosine similarity between two word vectors to see how semantically similar they are.Example: * (road, bus) $\approx$ 0.36(good, great) $\approx$ 0.892. Vector Arithmetic (Word Math)Performs operations like:$(\text{king} - \text{man} + \text{woman}) \rightarrow \text{nearest vector} = \text{queen}$$(\text{running} - \text{run} + \text{walk}) \rightarrow \text{walking}$3. Similar WordsReturns contextually similar words for a query.Example: * $\text{good} \rightarrow (\text{great, best})$$\text{king} \rightarrow (\text{monarch, prince, sultan etc.})$4. Sentence SimilarityMeasures semantic closeness between sentences by averaging/aggregating word vectors.Example: * "dog is a loyal animal" vs "cat is a domestic animal" $\rightarrow$ similarity $\approx$ 0.7 (moderate)5. Odd One OutDetects the least similar item in a list.Example: * (apple, banana, mango, car) $\rightarrow$ car6. Word ClusteringGroups words into K clusters by vector similarity.Search
